## Descrição

Desafio Concrete.

## Instalação

```bash
$ npm install
```

## Executando a API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Testes

```bash

# e2e tests
$ npm run test:e2e
```
